### Running

1. Clone this repository
2. Download raw data into your local project directory, following the directory layout encoded in NB1's dataset registry. The expected top-level structure is `<project_root>/Raw Data/{Thymus, Brain, Spatial transcriptomics, ...}/`
3. Set `MES_BASE_DIR` to point at your project root, or run from a directory that already contains the data folders:
```bash
   export MES_BASE_DIR=/path/to/your/project
```
4. Run notebooks sequentially (NB1 → NB6) in Jupyter:

```bash
git clone https://github.com/Sjtu-Fuxilab/innate.git
cd innate
pip install -r requirements.txt
export MES_BASE_DIR=/path/to/your/project
jupyter notebook
```

NB1 will fail at its `assert RAW_DIR.exists()` check if the data path is not set up correctly; downstream notebooks (NB2 onwards) consume processed AnnData objects from `<MES_BASE_DIR>/Process Data/` produced by earlier notebooks.
